# Recursion = problem ko apne jaise chhote problem me todna
aur khud pe bharosa rakhna ki chhota problem solve ho jaayega.

RECURSION KE 3 PILLARS 
1️⃣ Base Case

 Kab rukna hai
Agar base case nahi → infinite recursion 

2️⃣ Recursive Call

 Chhota problem kaun sa hai

3️⃣ Work (Before / After Call)

 Call se pehle ya baad kya kaam ho raha hai

Agar ye 3 clear hain → recursion clear hai.
